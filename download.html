Operating System Instructions
Windows Install the self-installing executable
Unix Get Vim through GitHub with
git clone https://github.com/vim/vim.git
Alternatively, get Vim through Mercurial with
hg clone https://bitbucket.org/vim-mirror/vim
A final possibility is an AppImage
Mac See MacVim for a GUI version and Homebrew for a terminal version.
